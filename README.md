# Простая Калькулятор

## Описание
Это приложение — простой калькулятор, который поддерживает основные операции: сложение, вычитание, умножение, деление и возведение в степень. Калькулятор также поддерживает оператор изменения знака и ввод чисел с десятичной точкой.

## Функции
- Сложение, вычитание, умножение, деление и возведение в степень.
- Поддержка ввода чисел с десятичной точкой.
- Изменение знака числа.
- Повтор последней операции.

## Установка
1. Склонируйте репозиторий:
   ```bash```
   `git clone <url>`
2. Откройте решение в Visual Studio.

3. Соберите и запустите проект.

## Использование
Вы можете вводить числа через кнопки или клавиши на клавиатуре. Используйте кнопки операций для выполнения вычислений, а кнопку "=" — для получения результата.
